## About

IT engineer at **Space Financial Group**, a U.S.-based holding with 4 companies (Virtual CFO, Staff Solutions, Taxes, LATAM Solutions). I work across two fronts: building the internal infrastructure that keeps the operation running, and delivering complete digital products for external clients.

**Internal / Space Financial Group**
- Built and own all 4 holding websites (Space Virtual CFO, Staff Solutions, Taxes, LATAM Solutions): design from scratch, custom CSS, responsive layouts, JSON-LD structured data, HubSpot form integration
- Space LATAM Solutions landing page: design, custom CSS, Meta Ads attribution, HubSpot + n8n lead capture flow
- Space LATAM Portal: full-stack client portal (React + Laravel + Cloudflare R2) — secure login, document management, full financial module (transaction management with classification/reconciliation/approval, chart of accounts, bank CSV import, P&L + Balance Sheet + Cash Flow with PDF/CSV export, QuickBooks integration, KPI dashboard), invoice builder, multilingual support (ES/PT/EN), CI/CD via GitHub Actions
- Full DNS ownership across all holding domains: DKIM/SPF/DMARC, MX, domain migrations, Google Domains
- Google Workspace admin (10+ users), DKIM/SPF/DMARC setup, email routing
- RevOps architecture: HubSpot + Meta Ads + WhatsApp attribution, Power BI dashboards, cold email pipelines across 120k+ leads
- HubSpot workflows, n8n automations, CRM architecture

**External clients (U.S. and LATAM)**
- Voice AI agent for Usar Biz: LiveKit + Grok/xAI, worker self-hosted 24/7 on a DigitalOcean Droplet (Docker + Ubuntu), multilingual lead qualification with n8n post-call automation
- Vibrantum Platform: Next.js + Firebase multilingual LMS and e-commerce (ES/EN/FR), 80+ SKUs, Hotmart, custom CMS
- Magic Metals and Quality Hands: production websites for U.S. clients
